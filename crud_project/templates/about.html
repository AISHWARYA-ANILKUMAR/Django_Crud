CRUD PROJECT

This project is a CRUD (Create, Read, Update, Delete) web application designed to manage student data. It allows users to add new student records, view the list of students, update existing information, and delete records when needed. The primary goal is to provide a simple interface for interacting with student information efficiently.

Features of the Project

  • Insert new student details.
  • View all student records.
  • Update existing student records.
  • Delete student records.

Technology Stack

  • Frontend: HTML, CSS (Bootstrap 5), JavaScript
  • Backend: Django (Python) for handling server-side logic and database operations
  • Database: SQLite (or any relational database)
